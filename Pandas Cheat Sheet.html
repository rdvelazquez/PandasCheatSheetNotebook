Summary Functions¶

Pandas provides a large set of summary functions that operate on different kins of panas objects (DataFrame columns, Series, GroupBy, Expanding and Rolling (see below)) and produce single values for each of the groups. When applied to a DataFrame, the result is returned as a pandas Series for each column
